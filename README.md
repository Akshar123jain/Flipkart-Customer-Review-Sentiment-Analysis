# Flipkart Customer Review Sentiment Analysis

## Project Overview

This project predicts customer sentiment from Flipkart product reviews using Machine Learning and Natural Language Processing (NLP).

The model classifies reviews into:

* Positive
* Negative
* Neutral

---

## Features

* Data Cleaning
* Text Preprocessing
* Sentiment Prediction
* Accuracy Evaluation
* Customer Review Testing
* Data Visualization

---

5. Predict Customer Sentiment
6. Evaluate Accuracy
